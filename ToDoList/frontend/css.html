🌟 CSS 简介

CSS（Cascading Style Sheets，层叠样式表）是用来描述HTML文档样式的语言。它控制网页的外观、布局和动画效果。

CSS 的作用

  • 样式控制：颜色、字体、大小等视觉效果
  • 布局设计：元素的位置和排列方式
  • 响应式：适配不同设备和屏幕尺寸
  • 动画交互：创建动态效果和用户交互
💡 学习提示： CSS就像是网页的化妆师，它能让普通的HTML变得美丽动人。掌握CSS是前端开发的核心技能！

📝 CSS 语法基础

CSS规则由选择器和声明块组成。声明块包含一个或多个声明，每个声明由属性和值组成。

/* CSS 基本语法 */
选择器 {
    属性: 值;
    属性: 值;
}

/* 示例 */
h1 {
    color: blue;
    font-size: 24px;
    text-align: center;
}

.highlight {
    background-color: yellow;
    padding: 10px;
    border-radius: 5px;
}

#header {
    width: 100%;
    height: 80px;
    background: linear-gradient(to right, #667eea, #764ba2);
}

🔍 语法解析

  • 选择器：指定要应用样式的HTML元素
  • 属性：要修改的样式特性
  • 值：属性的具体设置
  • 声明：属性和值的组合
  • 注释：使用 /* */ 包围

🎯 CSS 选择器

选择器用于选择要应用样式的HTML元素。CSS提供了多种选择器类型。

🏷️ 基本选择器

/* 元素选择器 */
p {
    color: #333;
}

/* 类选择器 */
.container {
    max-width: 1200px;
    margin: 0 auto;
}

/* ID选择器 */
#navigation {
    background-color: #f8f9fa;
}

/* 通配符选择器 */
* {
    box-sizing: border-box;
}

/* 属性选择器 */
input[type="text"] {
    border: 1px solid #ccc;
    padding: 8px;
}

input[placeholder] {
    font-style: italic;
}

🔗 组合选择器

/* 后代选择器 */
.container p {
    line-height: 1.6;
}

/* 子元素选择器 */
.menu > li {
    display: inline-block;
}

/* 相邻兄弟选择器 */
h2 + p {
    margin-top: 0;
}

/* 通用兄弟选择器 */
h2 ~ p {
    color: #666;
}

/* 多类选择器 */
.button.primary {
    background-color: #007bff;
    color: white;
}

📐 布局技术

🌊 Flexbox 布局

/* Flex 容器 */
.flex-container {
    display: flex;
    flex-direction: row; /* row, column, row-reverse, column-reverse */
    justify-content: center; /* flex-start, flex-end, center, space-between, space-around */
    align-items: center; /* flex-start, flex-end, center, stretch, baseline */
    flex-wrap: wrap; /* nowrap, wrap, wrap-reverse */
    gap: 20px;
}

/* Flex 项目 */
.flex-item {
    flex: 1; /* flex-grow flex-shrink flex-basis */
    flex-grow: 1;
    flex-shrink: 0;
    flex-basis: 200px;
    align-self: flex-start;
}

/* 实用示例 */
.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 1rem;
}

.card-grid {
    display: flex;
    flex-wrap: wrap;
    gap: 1rem;
}

.card {
    flex: 1 1 300px; /* 最小宽度300px，可伸缩 */
}

🏗️ Grid 布局

/* Grid 容器 */
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* 3列等宽 */
    grid-template-rows: auto 1fr auto; /* 3行 */
    grid-gap: 20px;
    
    /* 命名网格线 */
    grid-template-areas: 
        "header header header"
        "sidebar main main"
        "footer footer footer";
}

/* Grid 项目 */
.grid-item {
    grid-column: 1 / 3; /* 从第1列到第3列 */
    grid-row: 2 / 4; /* 从第2行到第4行 */
}

/* 使用命名区域 */
.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

/* 响应式网格 */
.responsive-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 1rem;
}

📱 响应式设计

响应式设计让网页在不同设备上都能完美显示。

📏 媒体查询

/* 基本媒体查询 */
@media screen and (max-width: 768px) {
    .container {
        padding: 10px;
    }
    
    .grid {
        grid-template-columns: 1fr;
    }
}

/* 多个断点 */
/* 手机 */
@media (max-width: 480px) {
    .text {
        font-size: 14px;
    }
}

/* 平板 */
@media (min-width: 481px) and (max-width: 768px) {
    .sidebar {
        width: 100%;
    }
}

/* 桌面 */
@media (min-width: 769px) {
    .container {
        max-width: 1200px;
    }
}

/* 高分辨率屏幕 */
@media (-webkit-min-device-pixel-ratio: 2) {
    .logo {
        background-image: url('logo@2x.png');
        background-size: 100px 50px;
    }
}

🎭 关键帧动画

/* 定义关键帧 */
@keyframes fadeIn {
    from {
        opacity: 0;
        transform: translateY(30px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}

@keyframes bounce {
    0%, 20%, 50%, 80%, 100% {
        transform: translateY(0);
    }
    40% {
        transform: translateY(-30px);
    }
    60% {
        transform: translateY(-15px);
    }
}

@keyframes rotate {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
}

/* 应用动画 */
.fade-in {
    animation: fadeIn 0.6s ease-out;
}

.bounce-element {
    animation: bounce 2s infinite;
}

.loading-spinner {
    animation: rotate 1s linear infinite;
}

/* 动画属性详解 */
.complex-animation {
    animation-name: fadeIn;
    animation-duration: 2s;
    animation-timing-function: ease-in-out;
    animation-delay: 0.5s;
    animation-iteration-count: infinite;
    animation-direction: alternate;
    animation-fill-mode: forwards;
    animation-play-state: running;
    
    /* 简写形式 */
    animation: fadeIn 2s ease-in-out 0.5s infinite alternate forwards;
}
